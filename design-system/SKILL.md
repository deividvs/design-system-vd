Key reminders for this brand:
- **Voice is Brazilian Portuguese**, plain and direct ("conversa de curral"). Speak to the rancher as **você**. Sentence case; UPPERCASE only for eyebrow labels and CTA buttons. No emoji.
- **Numbers are the brand** — set prices, stats and percentages in the DM Mono typeface and foreground them. The enemy is always *o achismo* (guesswork).
- **Color**: deep forest greens (primary), warm amber (the "money" accent — use sparingly), warm earth neutrals, cream/parchment backgrounds. No cold greys, no blue-purple gradients.
- **Type**: Bricolage Grotesque (display), Source Sans 3 (body), DM Mono (numbers).
- **Icons**: Lucide-style line icons only (24×24, stroke 2, round caps).

If the user invokes this skill without other guidance, ask what they want to build or design, ask a few clarifying questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.
